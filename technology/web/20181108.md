# HTML 5

###

HTML 5 文件包含DOCTYPE、標頭 (header) 與主體 (body) 等三個部分，
```
<!doctype html>
<html>

  <head>
    <meta charset="utf-8">
	<title>我的第一份HTML5文件</title>
  </head>	
  
  <body>
    <h1>Hello! HTML5!</h1>
  </body>
  
</html>

```
